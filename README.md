# MamaVege HR System

HR Management System for Mama Global International Sdn Bhd (MamaVege)

## Features
- Employee Management (onboarding, documents)
- Attendance (PIN + GPS/WiFi clock in/out)
- Leave Management (7 types, hourly calculation)
- OT Requests (supervisor approval)
- Claims (receipts, monthly limits)
- Payroll (EPF/SOCSO/EIS/PCB, PDF payslip, Excel export)
- Suggestion Box (anonymous, Director only)
- WhatsApp notifications

## Tech Stack
- Next.js 14 (App Router)
- Supabase (Database + Auth + Storage)
- Tailwind CSS
- Vercel (Deployment)

## Setup
1. Copy `.env.local.example` to `.env.local`
2. Fill in Supabase credentials
3. Run `npm install`
4. Run `npm run dev`
